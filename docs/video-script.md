## 一、录制前准备（5分钟）

### 浏览器设置
1. 打开 Chrome，访问 `hehexiake.github.io/maple-benefits`
2. 按 `Cmd + Shift + B` 隐藏书签栏
3. 按 `Cmd + -` 缩小一级，让界面更完整（或保持默认）
4. 窗口大小调整为约 **1200 × 750**，不要全屏
5. **清空档案数据**：进入"我的档案"，确认所有字段都是空的/未设置状态
   - 如果之前填过数据，打开浏览器控制台（`Cmd+Option+J`），输入 `localStorage.clear()` 回车，刷新页面

### 录制前停在这个画面
- 页面停在**看板首页**
- 所有数字显示 **0**（可领总金额 0，已申请 0）
- 这是视频的起点，制造"before"的感觉

## 四、制作步骤

### Step 1：录制屏幕（10分钟）

**工具：QuickTime Player（免费，Mac自带）**

1. 打开 QuickTime → 顶部菜单「文件」→「新建屏幕录制」
2. 点击录制按钮旁的下拉箭头 → 麦克风选「无」（不要录音）
3. 点录制按钮
4. **拖选录制区域**：只框住浏览器窗口，不要录进 Dock 和菜单栏
5. 按照脚本第二部分逐步操作
6. 录完后按 `Cmd + Ctrl + Esc` 停止录制
7. 保存为 `.mov` 文件

**操作节奏提示：**
- 填档案那段要**快而清晰**，每个下拉选完就立刻点下一个，不要犹豫
- 看到大数字跳出来后，手离开鼠标，**停够3秒再动**
- 滚动福利卡片时匀速慢滚，让每张卡片都能被看清楚

---

### Step 2：加字幕（15分钟）

**工具：剪映（免费）**

1. 打开剪映 → 「开始创作」→ 导入 `.mov` 文件
2. 时间轴上按脚本位置手动添加文字：
   - 点「文字」→「新建文字」
   - 拖动字幕块到对应时间段
3. 字体推荐：**PingFang SC Bold**（Mac自带，干净）
4. 颜色方案：
   - 普通字幕：白色 + 黑色描边（防止背景遮挡）
   - `$2,847/月` 那行：**黄色** `#FFD700`，字号放大到150%
5. 导出：1080P，MP4格式

**剪映小技巧：**
- 选中视频片段 → 右键 → 「变速」→ 填档案那段可以加速到1.3x，数字跳出那段减速到0.8x
- 最后加一秒黑底白字的链接卡（文字页面，不是字幕）
